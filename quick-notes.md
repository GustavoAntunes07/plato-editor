## Design system suggestions

- Visual direction: quiet, productive, readable, content-first desktop interface.
- Support balanced light and dark themes from the start.
- Base palette: graphite/gray neutrals with a discreet teal primary accent.
- Tertiary interaction color: sky blue (`#0EA5E9`), with `#0284C7` for light-theme focus rings and `#38BDF8` for dark-theme focus rings.
- Keep interaction roles separate: neutral surface hover, sky focus ring, teal selection border, and dedicated semantic colors.
- Recommended UI stack: shadcn/ui, Radix Primitives, Tailwind CSS, and Lucide React.
- Consider React Aria Components if advanced accessibility becomes the highest priority.
- Avoid adopting a strongly opinionated visual system such as Material UI, Ant Design, or Chakra as the foundation.
- Typography direction to refine: sans-serif UI, configurable reading font for Markdown, and monospace only for code.
- Define later: spacing scale, density, editor layout, component states, motion, elevation, and accessibility targets.
